The web can be a much better place

This course introduces the latest updates to HTML5 and CSS3. This is intended to be an easy introduction to get up-to-date with the current standard, but not a complete reference. You have the W3C for that.

What we do provide:

  • 600 slides
  • 30 labs
  • 90+ interactive examples embedded in the slides

A great opportunity for an interactive learning experience, the course is designed to be delivered in 24 hours including the labs. The slides are deliberately sketchy, since we strongly believe that Instructor Matters.
